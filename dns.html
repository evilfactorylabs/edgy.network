Pengujian

Lakukan query untuk domain coba.dong, jika jawabannya adalah 10.6.6.6 berarti kamu sudah menggunakan edgyDNS. Jika jawabannya adalah "bogus NXDomain", kemungkinan ISP kamu yang menjawab DNS Query tersebut.
