edgyDERP

Custom Tailscale™ DERP Servers

Konfigurasi(Access Controls)

{
  "derpMap": {
		"Regions": {
			"900": {
				"RegionID":   900,
				"RegionCode": "cgk",
				"RegionName": "Jakarta",
				"Nodes": [{
					"Name":     "1",
					"RegionID": 900,
					"HostName": "derp.edgy.network",
				}],
			},
		},
	},
}

Pelajari lebih lanjut tentang Tailscale DERP Servers disini.
